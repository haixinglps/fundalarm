**文件**: `FundServiceImpl.java`

**修复**:
```java
// 在 updatezhiying 方法的@Transactional事务中
// 遍历所有有持仓的档位，批量清空fene
for (Fund1Gaoduanzhuangbei2Ok position : duichongList) {
    // 只处理有持仓的档位（fene>0）

        continue;

    // 清空fene=0

    posUpdate.setZhiying(BigDecimal.ZERO);

    this.fund1Gaoduanzhuangbei2OkMapper.updateByExampleSelective(posUpdate, posExample);

```

### 4. 全平后清理T仓位Redis

**文件**: `DailyProfitTManager.java`

**新增方法**:
```java
public void clearAllPositionsRedis(String symbol) {
    // 清理所有T仓位的Redis记录
    for (TPosition pos : positions) {
        String posKey = "t:position:" + symbol + ":" + pos.id;
        jedisClient.del(posKey + ":zhang");
        jedisClient.del(posKey + ":okxPosId");
    }
    jedisClient.del(KEY_POSITIONS.replace("{symbol}", symbol));
}
```

## 防止重复仓位机制

| 场景 | 机制 | 时长 |
|------|------|------|
| T交易开仓 | Redis暂停标记 t:pause:{symbol} | 5分钟 |
| 底仓建仓 | Redis暂停标记 t:pause:{symbol} | 5分钟 |
| 数据库fene | 事务内批量清空所有档位 | 立即 |
| OKX接口 | close-position 一键全平 | 立即 |
| 分批止盈 | 清理batch:sold/batch:tp等Redis | 立即 |
